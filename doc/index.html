JASM Assembly Language

Overview

JASM is a simple assembly language targeting x86-64 Linux systems. It provides basic instructions for register operations, control flow, and system calls.

Getting Started

Installation

git clone https://github.com/jotrorox/jasm.git
cd jasm
make

Usage

jasm source.asm output
./output

Instructions

Labels

Labels are used to mark positions in code that can be jumped to. They are defined by adding a colon (:) after the label name. 

start:                  # Define a label named 'start'
    move rax, 42
loop:                  # Another label for looping
    # ... code here ...

Data Declarations

The data directive is used to declare variables and constants. It supports multiple formats: 

data message "Hello\n"      # String literal with escape sequences
data buffer size 1024      # Allocate buffer of given size
data value 0xFF           # Raw numeric value (hex)
data content file "input.txt"  # Include file contents

Move Instruction

The move instruction copies values between registers and memory. It supports multiple addressing modes: 

move rax, 42             # Load immediate value into register
move rax, rbx            # Copy between registers
move rax, [buffer]       # Load from memory
move [buffer], rax       # Store to memory

Jump Instructions

Jump instructions control program flow. They can be unconditional or conditional based on comparison results: 

jump label              # Unconditional jump
jumplt label           # Jump if less than
jumpgt label           # Jump if greater than
jumpeq label           # Jump if equal to

Compare Instruction

The comp instruction compares values and sets flags for conditional jumps: 

comp rax, 42           # Compare register with immediate
comp rax, rbx          # Compare two registers

# Example with conditional jump
comp rax, 0
jumpeq done           # Jump if rax equals 0

Add Instruction

The add instruction performs arithmetic addition: 

add rax, 5            # Add immediate to register
add rax, rbx          # Add register to register
                                
# Example counter increment
move rax, [counter]
add rax, 1
move [counter], rax

System Call

The call instruction triggers a system call. Register rax should contain the system call number, and other registers contain parameters: 

# Write to stdout example
move rax, 1           # sys_write
move rdi, 1           # stdout file descriptor
move rsi, message     # buffer address
move rdx, 5           # length
call                 # Execute system call

# Exit program example
move rax, 60          # sys_exit
move rdi, 0           # status code
call                 # Execute system call

Note: All numeric values can be specified in decimal (42), hexadecimal (0x2A), or binary (0b101010) format.

Data Directives

Supported Data Types

  • String literals: data label "string"
  • Buffer allocation: data label size N
  • File inclusion: data label file "filename"
  • Raw values: data label 0xFFFF

Examples

Hello World

# Hello World example
data msg "Hello, World!\n"

start:
    move rax, 1          # sys_write
    move rdi, 1          # stdout
    move rsi, msg        # message
    move rdx, 14         # length
    call                 # syscall
    
    move rax, 60         # sys_exit
    move rdi, 0          # status code
    call                 # syscall

Loop Example

# Count from 10 to 0
start:
    move rax, 10         # counter

loop:
    comp rax, 0          # check if counter is 0
    jumpeq done          # if zero, exit loop
    
    # ... do something ...
    
    move rax, rax-1      # decrement counter
    jump loop            # continue loop

done:
    move rax, 60         # sys_exit
    move rdi, 0          # status code
    call                 # syscall
